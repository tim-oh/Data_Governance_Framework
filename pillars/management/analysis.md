# Analysis

The Analysis building block covers the tools and processes used to understand, monitor, and audit data — including cataloguing what data exists, assessing data protection risks, and maintaining operational dashboards.

> _Placeholder: from framework text — comprehensive data catalogue, regular DPIAs for high-risk processing, active monitoring dashboards with named owners_
