# Part 4. Paths to data governance: Country cases

Policy implementation is a practical discipline that is best illustrated with real-world examples. This part highlights some of the steps that selected countries have taken, or are taking, in their data governance journey. The diversity of cases and stages of technical sophistication highlight that progress within the local context is what counts, that each country needs to find its own way on this policy path from where it finds itself at present, and that even the most progressive settings have ample space for policy development.
