# Print edition

This space contains the print/PDF edition of the Data Governance Framework — a single linear narrative assembled from the core content across the other spaces, with publication-grade front matter and transitions.

## Status

> _Placeholder: this space is assembled after the online content is finalised. The print edition is exported as a single PDF via GitBook's PDF export (requires Premium/Ultimate plan)._

## Structure

The print edition follows the structure of the original publication:

1. Front matter (cover, acknowledgements, table of contents, acronyms)
2. Introduction
3. Part 1: Framework overview
4. Part 2: Pillars of data governance
5. Part 3: Implementation guidance
6. Part 4: Country cases
7. Concluding remarks
8. References
9. Annexes (if any)

## Publication elements

- Version number and date
- How to cite — see [shared/how-to-cite.md](../shared/how-to-cite.md)
- Disclaimer — see [shared/disclaimer.md](../shared/disclaimer.md)
- Changelog / release notes
- Named maintainer and feedback route

## Content approach

The print edition shares approximately 80–90% of its content with the online spaces. Format-specific differences:

- **Online:** short intros, task-first navigation, cross-links, "related" blocks
- **Print:** linear narrative transitions, cover/front matter, citation/versioning, annexes
